# Interface Design System — Luxury Boutique

## Direction and Feel
- **Intent**: Calibration for a high-end collector. Calm, exclusive, tactile.
- **Atmosphere**: "Atelier Privado". 
- **Typography**: `Cormorant Garamond` (Hero/Editorial) and `DM Sans` (Precise Data).

## Depth Strategy
- **Layered & Soft**: Lean on subtle background color shifts (`Champagne` -> `Surface-1`) and whisper-quiet shadows.
- **Borders**: Using `rgba(0,0,0,0.05)` for invisible structure that becomes findable when needed.

## Color Palette
- **Base**: `#F9F7F2` (Champagne Base)
- **Ink**: `#1A1A1A` (Primary Text)
- **Gold/Flacon**: `#C8A96E` (Accents/Actions)
- **Velvet**: `#2D1B1B` (Emphasis/Headlines)
- **Mist**: `#8E8E8E` (Secondary/Metadata)

## Key Component Patterns
- **Sillage Profile**: Visual intensity indicator using 5-dot scale (Gold for active, Soft Border for inactive).
- **Cards**: "Manuscrito" style with `padding: 2.5rem` and subtle bottom border separation.
- **Buttons**: `letter-spacing: 0.1em`, uppercase, with smooth scale/rotate interactions.
- **Inputs**: Slightly darker background (`rgba(0,0,0,0.02)`) to signal "inset" area for content.
